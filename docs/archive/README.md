# AnimeStudio 文档存档

这里保存已经不属于当前默认 Library 主线的研究记录、单游戏审计和历史交接资料。

读取规则：

- 当前强制标准只看 [../PROJECT_EXPORT_STANDARDS.md](../PROJECT_EXPORT_STANDARDS.md)。
- 当前单游戏特殊适配看 [../game-profiles/README.md](../game-profiles/README.md)；存档里的单游戏审计不是当前 profile。
- 存档文档可以作为问题背景、失败案例或旧实现证据，但不能覆盖当前默认导出标准。
- 如果存档经验重新变成通用规则，应提炼后写回 `PROJECT_EXPORT_STANDARDS.md`，不要让规则散落在存档文档里。

## animation-research

旧动画求解、Unity bake、Humanoid/Muscle、Library Browser 动画预览和 Imported Avatar 诊断资料。动画当前不属于默认 Library 导出目标，这些资料只用于显式诊断或未来研究。

## game-audits

单游戏审计记录。它们可以说明某个游戏的已知边界，但不能作为通用 Unity 导出规则，除非已经被提炼到标准文档。

## history

项目迭代复盘和交接记录。用于理解为什么规则演化到当前状态，不作为最新标准。
